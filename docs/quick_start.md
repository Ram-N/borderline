

## README checklist (quick start)

* Install & run

  ```sh
  npm i
  npm run dev
  ```
* Add first set:

  * Place edited map in `public/images/maps/<region>/<file>.png`
  * Create `/public/data/questions/<region_country_set>.json`
  * Add entry to `/public/data/questions.index.json`
* Start the quiz from Home → select your new set
